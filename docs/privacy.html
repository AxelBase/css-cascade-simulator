Privacy Policy

Privacy Policy

Effective Date: December 3, 2025

At CSS Cascade Simulator, your privacy is our highest priority. This tool was built from the ground up to be completely private and secure by design. We believe you should understand exactly how the CSS cascade works without ever compromising your personal information or creative work.

No Data Collection Whatsoever

We do not collect, store, or transmit any data — period. Every CSS rule you write, every selector you test, and every simulation you run happens entirely in your browser. Nothing is sent to our servers, third parties, or cloud services. Your stylesheets never leave your device.

How the Tool Works

All processing occurs locally using JavaScript running in your browser. When you input CSS rules and a target element, the specificity calculation, selector matching, and winner determination are performed instantly on your computer. There is no backend, no database, and no logging. Even if you refresh the page, your input is gone — because we never had it in the first place.

No Tracking or Analytics

We do not use cookies, localStorage for tracking, fingerprinting, Google Analytics, Meta pixels, or any monitoring tools. There are no advertising partners, no behavioral tracking, and no session recording. You remain completely anonymous at all times.

Third-Party Content

This site uses only Bootstrap 5 via CDN for styling and Bootstrap Icons for visual clarity. These are static files that do not track users or set cookies. No external scripts run beyond what is necessary for the simulator to function.

Your Data Stays Yours

Because everything runs client-side, you can safely test proprietary stylesheets, internal design systems, or confidential UI code. Whether you're a solo developer, a large team, or an educator, your work remains 100% private and secure.

Updates to This Policy

We may update this Privacy Policy if new features are added. Any changes will be clearly posted here with the updated effective date. We encourage you to review it periodically.

Contact

Questions? Reach out via the GitHub repository linked in the header.

Thank you for trusting a tool that respects your privacy as much as you respect great CSS.
